# Handwritten Digit Recognition (MNIST)

## Project Overview
This project aims to implement a Bayes Classifier or K-Nearest Neighbors (KNN) algorithm to recognize handwritten digits from the MNIST dataset.

## Deliverables

### Use Case Summary
A summary outlining the problem, approach, and expected outcomes.
*(To be placed in the `docs/` directory)*

### Slide Presentation
A 5 to 10-minute slide presentation summarizing the project.
*(PDF version to be placed in the `presentations/` directory)*

### Public Git Link
Link to the version control repository for this project.

### Code
The implementation of the chosen classification algorithm (Bayes Classifier or KNN) for MNIST digit recognition.
*(To be placed in the `notebooks/` or `src/` directory)*

## Data
The MNIST dataset will be used for training and testing.
